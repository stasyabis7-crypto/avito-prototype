# Avito — Передача вещей: UX Prototype

A clickable HTML prototype for UX research, built from Figma designs.

## Pages

- **index.html** — Main landing page (all 8 sections)
- **campaign.html** — Campaign page: «Передача вещей для акции Помощь семьям»

## Sections (Landing)

1. Hero — with CTA scrolling to "Как передать вещи"
2. Как передать вещи — Interactive tabs (4 options)
3. Как это работает — 3 static cards
4. Что можно передать — 3 cards
5. Informational banner — "Объявление не будет видно в поиске"
6. Что происходит после размещения — 3 cards
7. Активные акции — 2 campaign cards (clickable → campaign.html)
8. Вопросы и ответы — Accordion FAQ

## Interactive behaviors

- Scroll to sections from buttons
- Tab switching in "Как передать вещи"
- FAQ accordion (one open at a time)
- External links open in new tab
- Campaign cards navigate to campaign page

## Deploy to Vercel

1. Upload this folder to a GitHub repo
2. Connect to Vercel
3. Deploy as Static Site (no build command needed)

Or drag-and-drop the folder at https://vercel.com/new

## Local preview

Simply open `index.html` in a browser — no server needed.
